# `awsinfo efs mount-targets [file-system-filter]* -- [mount-target-filter]*`

List all Mount Targets for an EFS file system.

## First filter matches against

* FileSystemId
* LifeCycleState
* Name
* PerformanceMode
* ThroughputMode
* Encrypted


## Second filter matches against

* MountTargetId
* SubnetId
* LifeCycleState
* IpAddress
* NetworkInterfaceId
